# RULE-001: 记住指令

## 触发条件

当用户说出 **"记住"** 或其同义表达（如 "记下来"、"牢记"、"写入记忆"、"记住这个"）时触发。

## 执行动作

1. 提取用户要求记住的具体内容。
2. 将内容以结构化方式写入 `.trae/rules/` 或 `.trae/memory/` 目录下的文件，文件名格式：`RULE-{序号}-{简短描述}.md`。
3. 告知用户已写入的位置。

## 示例

- 用户："记住，以后所有 SQL 都用 PostgreSQL 方言"
  → 写入 `.trae/rules/RULE-002-sql-default-postgres.md`
- 用户："把这段代码记下来"
  → 提取代码内容，写入 `.trae/memory/` 对应文件
