## army-oracle

This module provides Oracle dialect-specific criteria API.

### Main Packages

#### io.army.criteria.oracle

Oracle-specific criteria interfaces for building SQL statements.

- **`OracleStatement`**: Base interface for Oracle statements.
- **`OracleQuery`**: Build SELECT queries with Oracle-specific features.
- **`OracleWindowBuilder`**: Build window functions with Oracle syntax.

### Oracle-Specific Features

- **Window Functions**: Oracle window function support.
- **Oracle Query**: Oracle-specific SELECT query features.

### Dependencies

- **`army-core`**: Core criteria API and type mappings.
