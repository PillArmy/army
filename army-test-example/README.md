## army-test-example

Parent module for Army test example applications.

### Submodules

- **army-example-stock-data**: Stock domain data layer with DAO and service implementations
- **army-example-stock**: Stock web application with AI-powered chat functionality
- **army-type-test**: Type mapping tests for PostgreSQL advanced types (array, composite, hstore, XML)

### Purpose

Demonstrates real-world usage of Army framework with Spring integration, showing:

- Domain modeling with annotations
- Metamodel generation at compile time
- Service and DAO layer patterns
- Programmatic transaction management
- Integration with Spring AI for AI features
