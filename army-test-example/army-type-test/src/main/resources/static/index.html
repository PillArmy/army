API 端点

POST /api/postgre-types - 创建新的类型测试记录
PUT /api/postgre-types - 更新类型测试记录
GET /api/postgre-types/{id} - 根据ID获取记录
GET /api/postgre-types/first - 获取第一条记录
POST /api/postgre-types/demo - 创建演示数据
